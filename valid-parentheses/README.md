# valid-parentheses

## 问题
输入一个只包含`(`、`)`、`{`、`}`、`[`、`]`的字符串，判断是否为合法字符串。   
合法的判定条件：某个符号以正确顺序闭合

## 思路
典型的栈操作。遍历字符串的每个字符，以符号左半部`(`、`{`、`[`为基准，，先出现的必然后闭合，所以遍历到左半部字符时，对应将右半部`)`、`}`、`]`压入栈，如果是右半部的字符，则弹出栈顶元素，如果栈顶元素与当前字符不等，则说明闭合顺序不对。若最终栈为空，则为有效字符串
